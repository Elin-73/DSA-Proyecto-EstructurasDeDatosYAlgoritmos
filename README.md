El ejecutable se encuentra en dist/main con nombre main.exe.
Al ejecutar el programa se abriran dos ventanas,
  la ventana principal tiene por nombre "Projecto Final: ...",
  cerrar una ventana cierra ambas, y por lo tanto, el programa entero.
No es necesario descargar los requisitos a menos que
  se quiera construir y/o modificar el proyecto.

Al ejecutar el proyecto se verán 6 pestañas en la parte derecha de la ventana,
  cada una representa una estructura de datos:
1-. Pila
2-. Cola
3-. Arreglo
4-. Lista Circular Simple
5-. Árbol Simple
6-. Grafo con recorrido de Kruskal

Cada ventana incluye su propio espacio en donde se tendrá que hacer
  click para escribir lo que se quiera agregar y usar el teclado
  para introducir la información.
Una vez puesta la información dentro de las cajas de manera
  correspondiente, haga click en los botones para
  introducir o quitar dicha información, o modificar la estructura.
